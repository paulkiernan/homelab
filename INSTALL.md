# Arch Installation Pre-reqs

The following packages need to be installed before these script will work.

## Arch Linux

```bash
yay -S \
    aur/flux-bin \
    community/age \
    community/direnv \
    community/ipcalc
```

## MacOS

```bash
brew install \
    age \
    direnv \
    fluxcd/tap/flux \
    go-task/tap/go-task \
    ipcalc
```
